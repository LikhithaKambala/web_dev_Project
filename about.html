About This Page

This web page is a simple interactive project created using HTML, CSS, and JavaScript. First, you can register by entering your name and email. Once registered, you can explore a fun feature: fetching Pokémon data directly from the Pokémon API! You will see Pokémon names, images, height, and weight displayed dynamically.

This project demonstrates:

  • Form handling and conditional display
  • API fetching using JavaScript
  • Dynamic content rendering on the same page
